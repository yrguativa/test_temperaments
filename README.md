# Test de Temperamentos 🧠

Aplicación Next.js para descubrir tu temperamento dominante según los 4 temperamentos clásicos: Sanguíneo, Colérico, Melancólico y Flemático.

## Rutas

| Ruta | Descripción |
|------|-------------|
| `/` | Test principal con 40 preguntas desde JSON, opciones en cards con aspect ratio, animaciones y gradientes |
| `/old-test` | Versión anterior con 10 preguntas desde el store |

## Stack

- **Next.js 16** — App Router
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Zustand** — Estado global
- **Recharts** — Gráficos de resultados

## Desarrollo

```bash
npm run dev
# Abrir http://localhost:3000
```

## Build

```bash
npm run build
npm start
```
